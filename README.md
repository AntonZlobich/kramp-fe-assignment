## Getting Started

### Prerequisites

- Node.js 24+
- npm

### Installation

```bash
npm install
```

### Running the Application

To run both the frontend and backend:

```bash
npm run dev
```

### Running Tests

```bash
npx nx test webshop
```

### Optional: To run Storybook

```bash
npm run storybook
```
